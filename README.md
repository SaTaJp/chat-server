# Chat Server

Python の `socket` と `threading` を使って作成した、シンプルな CUI チャットアプリです。  
サーバーに複数のクライアントが接続し、参加者全員にメッセージを配信できます。

## 構成

- `server.py`
  チャットサーバー本体です。クライアント接続を受け付け、ユーザーごとの ID を割り当てて、メッセージを全員に配信します。
- `client.py`
  チャットクライアントです。サーバーに接続し、入力したメッセージを送信しながら、別スレッドで受信メッセージを表示します。

## 動作概要

- サーバーは `0.0.0.0:5050` で待ち受けます。
- クライアント接続時に `User1`, `User2` のような ID が自動で割り当てられます。
- 送信されたメッセージは接続中の全クライアントへブロードキャストされます。
- ユーザーの参加・退出時にはサーバー通知が全員に送られます。
- `quit` と入力するとクライアントは切断されます。

## 実行方法

### 1. サーバーを起動

```bash
python3 server.py
```

起動すると以下のように表示されます。

```text
Server started on port 5050
```

### 2. クライアントを起動

別ターミナルで以下を実行します。

```bash
python3 client.py
```

複数人で試したい場合は、同じマシン上で複数のクライアントを起動できます。

## 利用例

クライアント起動後、メッセージを入力して Enter を押すと送信されます。

```text
Hello
```

終了する場合は以下を入力します。

```text
quit
```

## 注意点

- 現在の `client.py` は接続先を `127.0.0.1` に固定しているため、同一マシン上のサーバー接続を前提としています。
- 別の PC から接続する場合は、`client.py` の `HOST` をサーバーマシンの IP アドレスに変更してください。
- 例外処理は簡易的なため、本番用途ではなく学習用・動作確認用の実装です。
